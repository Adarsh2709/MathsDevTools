Theory & Notes

• Prime Distribution: π(n) ≈ n / ln n (Prime Number Theorem) gives an estimate of how many primes ≤ n.
• Sieve of Eratosthenes: mark multiples of each prime starting from 2; segmented sieves reduce memory for large ranges.
• Practical Tips: generate base primes up to √hi, then mark in blocks [lo, hi] to get primes in range efficiently.
• Applications: cryptography (keys), hashing, random sampling, modular arithmetic problems.
