Theory & Notes

• Digits: bases 2–36 use 0–9 then A–Z. Value of a string is Σ di·basei.
• Conversion: parse to decimal, then repeatedly divide by target base for integer part; multiply fractional part by base to extract digits.
• Fractions: many decimals become repeating in other bases; we cap fractional digits for practicality.
• Common: binary (2), octal (8), decimal (10), hex (16) — useful in systems and low-level coding.
